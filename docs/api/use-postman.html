# Use Postman with the SalesTim API

ABSTRACT
You can use the SalesTim API Postman collection to get started with the SalesTim API in minutes.
This article explains how to get up and running with Postman and SalesTim (You can also explore SalesTim APIs directly in your web browser by using API Explorer).

TABLE OF CONTENTS

  • How to import the collection

# How to import the collection

To import the Postman collection:

  1. Download and register for Postman (opens new window).
  2. Choose File | Import ....
  3. Select Import From Link.
  4. Paste the following URL and choose Import.
    5. https://docs.nbold.co/api/production/definitions/open-api/apiDefinition.swagger.yaml

https://docs.nbold.co/api/production/definitions/open-api/apiDefinition.swagger.yaml

You are now up and running with the SalesTim API collection.
