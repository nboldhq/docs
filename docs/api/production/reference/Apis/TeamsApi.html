# TeamsApi

All URIs are relative to https://api.salestim.io/api/v1.0

Method HTTP request Description
addTeamMember POST /teams/{teamId}/members Add a team member
archiveTeam POST /teams/{teamId}/archive Archive a team
createTeamChannel POST /teams/{teamId}/channels Create a new team channel
createTeamChannelTab POST /teams/{teamId}/channels/{channelId}/tabs Create a new team channel tab
createTeamProvisioningJob POST /teams/provisioning Create a new team based on a template
deleteTeam DELETE /teams/{teamId} Delete a team
getTeam GET /teams/{teamId} Get a team
getTeamChannelTabs GET /teams/{teamId}/channels/{channelId}/tabs Get team channel tabs
getTeamChannels GET /teams/{teamId}/channels Get team channels
getTeamPrimaryChannel GET /teams/{teamId}/channels/primary Get the primary channel of a team
unarchiveTeam POST /teams/{teamId}/unarchive Unarchive a team
updateTeam PATCH /teams/{teamId} Update a team
